# AutoSplit GIEEE v1.2.0 リリースノート

## ✨ 新機能

- **今日の記録 (Today's Split Logger)**:
  - 日々の練習成果を自動でCSVファイルに記録できるようになりました。
  - **記録されるデータ**:
    - **区間タイム**: 前回のSplitからの経過時間
    - **ロード時間**: 区間内で検知されたロード時間の合計
  - **使い方**:
    - 設定画面の「監視設定」タブで「その日の記録 (CSV)」をONにします。
    - 監視を開始すると、`YYYYMMDD_HHMMSS_todays_split.csv` というファイルが作成されます。
    - 監視を開始（セッション開始）するたびに新しいファイルが作られるので、管理が簡単です。

- **監視間隔の高速化対応**:
  - 設定画面で監視間隔を最短 **1ms** まで設定できるようになりました（以前は16ms制限）。
  - より高速な検知が必要な場合に有効です。
  - なお、画像検知ですでに100msほどかかっているため、50ms未満の値に設定することをお勧めします。

## 🐛 バグ修正

- **設定が勝手にリセットされる問題を修正**: ターゲットウィンドウ（例：原神）が起動していない状態で設定画面を開いて保存すると、ターゲット設定が「フルスクリーン」に上書きされてしまう問題を修正しました。
  - 今後は、ウィンドウが見つからない場合でも設定が維持されます（「(未検出)」として表示）。
  - これにより、ターゲットウィンドウが起動していない状態で監視を開始しようとした際に、正しく「ウィンドウが見つかりません」というエラーが表示されるようになります。
